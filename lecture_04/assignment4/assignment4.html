# How does the sample size and the number of simulations affect the distribution of the sample means in the code below?

# As the sample size increases, the distribution of sample means becomes narrower and more normal. 
# Smaller sample sizes create a high variance (wide spread). 
# Large sample sizes create a low variance (concentration is closer to the true mean).

# Increasing the number of simulations makes the histogram smoother. 
# If the number of simulations is too low, the histogram will look irregular.
# Therefore, you need a sufficient number of simulations in order to make sure that the histogram actually represents the distribution.
